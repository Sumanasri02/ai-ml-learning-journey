# Orientation

## Overview
This module gives a basic understanding of Artificial Intelligence and
Machine Learning. It helps learners understand core concepts before
starting mathematical and coding-based topics.

---

## Objectives
- Understand what AI, ML, and Deep Learning are
- Learn the basic Machine Learning workflow
- Get clarity on common AI/ML roles

---

## Topics Covered
- AI vs Machine Learning vs Deep Learning
- Machine Learning lifecycle
- AI/ML roles in industry

---

## Why This Module
Starting AI/ML without understanding the basics often leads to confusion.
This module ensures a clear foundation before moving to implementation.

---

## Outcomes
After completing this module, learners will be able to:
- Explain AI, ML, and DL clearly
- Understand how an ML project flows from start to end
- Identify different AI/ML roles
